# RESEETA: Converting Cursive Handwritten Medical Prescriptions into Digital Text Using VIT-CRNN-Based Deep Learning Architecture

A comprehensive software and deep learning system utilizing Vision Transformers (ViT) and Convolutional Recurrent Neural Networks (CRNN) to accurately process and digitize cursive medical prescriptions. Built on the Laravel framework, this system provides a seamless interface for medical data extraction and management.

## 📖 Overview

Deciphering handwritten medical prescriptions is a critical challenge in healthcare administration. This thesis project bridges advanced computer vision with an accessible web portal, ensuring that complex cursive text is efficiently translated into secure, digital formats.

## 🚀 How to Run Locally

**1. Navigate to the Directory:**

```bash
cd reseeta-vit-crnn
```

**2. Install Dependencies:**

```bash
composer install
npm install
```

**3. Environment Setup:**
Duplicate the .env.example file, rename it to .env, and configure your local database settings.

**4. Launch the Server:**

```bash
php artisan serve
```
